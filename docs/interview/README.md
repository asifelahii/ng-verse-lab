# Interview Prep (Angular Jr SWE)

## How to use

- Each topic has: concepts → common questions → mini answers → pitfalls.
- Every week: 10 questions + 1 mock review.

## Topics

- TypeScript (core + generics + narrowing)
- Angular Fundamentals (components, templates, DI)
- RxJS (observables, operators, subjects)
- Routing (guards, resolvers, lazy loading)
- Forms (reactive forms, validators)
- HTTP (interceptors, error handling, caching)
- Change Detection (default vs OnPush)
- Testing (Jasmine/Jest concepts, TestBed, HttpTestingController)
- Architecture (modules, feature boundaries, smart/dumb components)
